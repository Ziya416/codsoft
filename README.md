CodSoft Data Science Internship 

This repository contains the tasks completed during the Data Science internship at CodSoft. The objective of these projects is to explore data analysis, preprocessing, and machine learning modeling techniques using Python. 

Tasks Completed
Task 1: Titanic Survival Prediction 

Built a machine learning model that predicts whether a passenger on the Titanic survived or not. 

Utilized a dataset containing individual passenger information, such as age, gender, ticket class, fare, and cabin details. 

Task 3: Iris Flower Classification 

Developed a model to classify iris flowers into three distinct species: setosa, versicolor, and virginica. 

Trained the model to learn from and categorize the flowers based on their sepal and petal measurements. 

Task 4: Sales Prediction Using Python 

Created a model to forecast the amount of a product that customers will purchase. 

Analyzed various factors that influence sales, including advertising expenditure, target audience segmentation, and advertising platform selection.
